# 城镇任务列表
每次重置城镇，在完成 3 个城镇入门指南任务后，将开放非城镇内容的任务。可点击跳过任务自动开放，但浙江无法获得被跳过的任务的奖励。  
商店-城镇标签出售的，增加指定技能经验的装备，需要完成一定数量城镇任务后解锁，这包括城镇入门指南任务。

## 图示
此任务列表不会按难度划分任务，而是依据任务类型。  
任务难度分别以 A-E 前缀标注，它们有固定的 GP 报酬：
* A - 简单 - 5千
* B - 普通 - 25千
* C - 困难 - 500千
* D - 极难 - 5百万
* E - 精英 - 10百万

## 交付类任务
